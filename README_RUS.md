# Задача коммивояжёра

## Задание
Найдите самый дешёвый способ посетить все города и вернуться в исходную точку.

![TSP_graph](img/TSP_graph.png)  
Граф с конечным числом точек и стоимостью проезда между каждой парой. Стоимость проезда симметрична (между двумя узлами есть прямой и обратный пути). Цель — найти гамильтонов цикл с минимальным общим весом.

Создайте таблицу с именованными узлами, используя структуру {point1, point2, cost}, и заполните данные на основе рисунка.  
Напишите SQL-запрос, который возвращает все туры с минимальной стоимостью проезда, если начать с города «A».  
Пример выходных данных:  
| total_cost | tour |
| ------ | ------ |
| 80 | {a,b,d,c,a} |
| ... | ... |  

Сортируйте данные по total_cost, а затем по tour.
